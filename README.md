# ReverseFPMC

### Compile:
```
cmake .
make
```

### Usage:
```
./ReverseFPMC <username> <host> <port>
```
Sometimes it dosnt work. You have to rejoin and try it again.
When you joined wait 5 seconds and then run the programm.
